1. Users

    id (PK)

    name

    email (unique)

    password (hashé)

    phone (nullable)

    email_verified_at (nullable)

    id_role (FK → Roles, default: 1)

    id_service (FK → Services, nullable)

    created_by (FK → Users, nullable)

    api_token (nullable)

    remember_token

    created_at

    updated_at


2. Roles

    id (PK)

    name (citoyen, agent, admin, responsable)

    created_at

    updated_at


3. Services

    id (PK)

    name

    description (nullable)

    image (nullable)

    responsable_id (FK → Users, nullable)

    created_at

    updated_at


4. Plaintes

    id (PK)

    code_suivi (unique, nullable)

    id_user (FK → Users, nullable)

    id_service (FK → Services, nullable)

    agent_id (FK → Users, nullable)

    title

    description

    piece_jointe (nullable)

    contact_nom (nullable)

    contact_info (nullable)

    statut (Enregistrée, En cours, Résolue, Rejetée)

    priorite (normale, haute, urgente)

    created_at

    updated_at


5. Réponses

    id (PK)

    agent_id (FK → Users)

    complaint_id (FK → Plaintes)

    message

    created_at

    updated_at


6. Notifications

    id_notification (PK)

    id_user (FK → Users)

    id_plainte (FK → Plaintes)

    type (email, SMS, in-app)

    content

    status (sent, read)

    created_at

    updated_at


7. Feedback

    id_feedback (PK)

    id_plainte (FK → Plaintes)

    id_user (FK → Users)

    note (1-5)

    comment (nullable)

    created_at

    updated_at


8. Logs

    id (PK)

    user_id (FK → Users)

    ip_address

    action

    details (nullable)

    created_at

    updated_at


9. Statistiques

    id_stat (PK)

    id_service (FK → Services)

    nombre_plaintes

    temps_moyen_resolution (nullable)

    taux_satisfaction (nullable)

    created_at

    updated_at


10. Rapports

    id (PK)

    agent_id (FK → Users)

    service_id (FK → Services)

    responsable_id (FK → Users)

    title

    content

    file_attachment (nullable)

    is_read (default: false)

    created_at

    updated_at


11. Tables système Laravel

    password_reset_tokens

    sessions

    cache
