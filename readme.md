# Application web de recherche et de reservation des produits locaux

# Stack Developpement , Ops and Features

### Developpement

- React js / SPA
- laravel / Rest full api 
- Postgres / SGBD
- Redis / Cache



### CI / CD  and Container

- github action
- docker

### Features 

- Account CRUD
- Auth (google,man auth) or Passwordless 
- Otp mail or sms
- Search and Filters 
- Detailed Product Sheet
- Shopping Card and  Reservation system
- Generating a reservation ticket
- User  History
- Rating system 
- Sailer Dashboard
- Inventory management
- Reservation management 
- Stats
- Notification web , whatsaap 
- Reservation miner
- Anti-spam System
- Location system
